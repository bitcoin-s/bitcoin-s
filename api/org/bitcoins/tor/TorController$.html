  32.  def setUpHiddenService(controlAddress: InetSocketAddress, authentication: Authentication, privateKeyPath: Path, virtualPort: Int, targets: Seq[String] = Seq.empty)(implicit system: ActorSystem): Future[InetSocketAddress]

    Set up a Tor hidden service.

    Set up a Tor hidden service. It expects a Tor daemon is up and running, and listening on the control address.

    Note: only Tor protocol v3 is supported

    controlAddress

    Tor control address

    authentication

    Tor controller auth mechanism (password or safecookie)

    privateKeyPath

    path to a file that contains a Tor private key

    virtualPort

    port for the public hidden service (typically 9735)

    targets

    address of our protected server (format [host:]port), 127.0.0.1:virtualPort if empty
