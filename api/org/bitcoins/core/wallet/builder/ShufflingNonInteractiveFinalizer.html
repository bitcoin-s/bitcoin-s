Companion object ShufflingNonInteractiveFinalizer

case class ShufflingNonInteractiveFinalizer(inputInfos: Vector[InputInfo], feeRate: FeeUnit, changeSPK: ScriptPubKey) extends RawTxFinalizer with Product with Serializable

A finalizer which adds a change output, performs sanity checks, shuffles inputs and outputs, and adds non-signature witness data. This is the standard non-interactive finalizer within the Bitcoin-S wallet.
