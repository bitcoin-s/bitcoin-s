  25.  case object Broadcasted extends DLCState with Product with Serializable

  26.  case object Claimed extends DLCState with Product with Serializable

  27.  case object Confirmed extends DLCState with Product with Serializable

  28.  case object Offered extends DLCState with Product with Serializable
