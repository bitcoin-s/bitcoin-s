  5.  def blockWeight: Long

    This is the new computation to determine the maximum size of a block as per BIP141 https://github.com/bitcoin/bips/blob/master/bip-0141.mediawiki#block-size The weight of a block is determined as follows:

    This is the new computation to determine the maximum size of a block as per BIP141 https://github.com/bitcoin/bips/blob/master/bip-0141.mediawiki#block-size The weight of a block is determined as follows:

    Base size is the block size in bytes with the original transaction serialization without any witness-related data

    Total size is the block size in bytes with transactions serialized as described in BIP144, including base data and witness data.

    Block weight is defined as Base size * 3 + Total size https://github.com/bitcoin/bitcoin/blob/7490ae8b699d2955b665cf849d86ff5bb5245c28/src/primitives/block.cpp#L35
